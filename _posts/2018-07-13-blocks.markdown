# 電梯
![](/assets/img/wiki/elevator/overview.png)
### 合成
使用合成台來合成(無序)
![](/assets/img/wiki/elevator/recipe.png)
### 用途
使玩家在電梯之間快速移動。
### 功能
電梯(至少兩個)垂直放置時，蹲下可以向下傳送，跳躍可以向上傳送。
### 其他
電梯的最遠距離是8格方塊(不包括電梯自己上下8格)。
